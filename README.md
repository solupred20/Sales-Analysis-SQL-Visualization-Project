# Sales Analysis — SQL & Visualization Project

## Overview

This project analyzes product-level sales data using SQL and presents the results through a clear visualization. The goal is to extract key insights about revenue distribution across products and support data-driven decisions.

## Project Structure

sales-analysis-project/

* sales_data.db
* sales_summary.csv
* sales_chart.png

## Data Processing

* Queried sales data from a SQLite database
* Aggregated total quantity and revenue per product
* Exported results into a structured CSV file

## Key Metrics

* Total quantity sold per product
* Total revenue per product

## Visualization

### Revenue by Product

* Bar chart comparing revenue across products
* Clearly shows top-performing product (Laptop)
* Highlights lower-performing items like Mouse and Monitor

## Insights

* Laptop generates the highest revenue by a significant margin
* Headphones and Monitor show moderate performance
* Mouse has the lowest revenue despite reasonable quantity, indicating lower pricing
* Revenue distribution is uneven, suggesting dependency on a single high-performing product
